# Example - Logging Serial to PC
This example shows how it is possible to capture data logged to the serial monitor by the Arduino with a Python script running on the PC.

## Hardware
```
+------+
|  PC  |
+------+
    |
    |  serial    +-----------+
    +------------|  Arduino  |
                 +-----------+
```
